### Quick Glossary  

LANraragi uses a lot of specific terms that might be confusing at first glance.  
This quick reference is meant to help you out if you don't want to read the docs and try to jump in straight away.  

- **Archive**: The base unit of content within LANraragi. AKA a book, a manga, whatever.  
- **Chapter**: A delineation within an Archive that begins and ends at a specific page.  
- **Tag**: Comma-separated metadata attached to an Archive.  
- **Index**: The main page of LANraragi where you search and open Archives. AKA your Library.  
- **Category**: A collection of Archives that can be displayed in the Index. Can either be static or dynamic.  
- **Bookmark**: A quick way to "favorite" an Archive by adding it to a dedicated "Favorites" Category.  
- **Stamp**: Geo-positioned metadata (X/Y coordinates) attached to a page within an Archive.  
- **Tankoubon**: A merged set of Archives with custom order that behaves as a single unit in the Index. AKA Meta-Archives.
